# Code of Conduct

We want this repository to be a practical place for technical collaboration. The expectations below apply to issues, pull requests, discussions, and any community space connected to the project.

## Expected Behavior

- Be respectful and concise.
- Keep feedback focused on ideas, code, and behavior.
- Assume good intent before escalating a disagreement.
- Help others learn by sharing context instead of gatekeeping.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Threats, intimidation, or doxxing.
- Spam, trolling, or repeated derailment of technical threads.
- Any conduct that makes participation unsafe or exclusionary.

## Reporting Concerns

If you encounter behavior that violates this code of conduct, use a private channel to contact the maintainers or open a confidential discussion with the repository team.

## Enforcement

Project maintainers may remove comments, close issues, or restrict participation when needed to preserve a safe and productive environment.
